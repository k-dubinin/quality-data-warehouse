
# КУРСОВАЯ РАБОТА 
## по теме «Реализация хранилища данных для системы управления качеством продукции»
## Выполнил Дубинин Константин Андреевич 
## группа ИД22-4

Данный проект представляет собой реализацию хранилища данных (Data Warehouse) для анализа качества продукции на производстве.  
Проект выполнен в рамках курсовой работы и включает полный цикл: подготовку данных, создание хранилища, загрузку данных и выполнение аналитических запросов.
---

## Цель проекта

Целью проекта является разработка хранилища данных на основе реляционной СУБД PostgreSQL, позволяющего выполнять аналитическую обработку информации о качестве продукции с использованием OLAP-операций и оконных функций.

---

## Архитектура решения

Хранилище данных реализовано по схеме "звезда" и содержит:

- staging-слой для загрузки исходных данных;
- таблицы измерений (dimensions);
- таблицу фактов (fact);
- аналитические SQL-запросы.

Используемые технологии:
- PostgreSQL
- Python
- SQL (DDL, DML, OLAP)
- pandas
- psycopg2
